# AssignItApp

# Pre-reqs
- Install [Node.js](https://nodejs.org/en/)

# Getting started
- Clone the repository
```bash
$ git clone https://github.com/JacielG/assign-it-app/
```
- Install dependencies
```bash
$ cd assign-it-app
$ npm install
```
- Run App on Dev Server
```bash
$ ng serve --o
```
Navigate to `http://localhost:4200/`
